üîß Customization Guide
Updating Personal Information
Contact Details: Update contact information in the Contact page section

Experience: Modify the timeline items in the Experience section

Skills: Adjust skill categories and levels in the Skills section

Certifications: Update the certifications array in the JavaScript section

Changing Images
Gallery Images: Replace Unsplash URLs in the galleryImages array in the JavaScript section

Hero Section: Consider adding a professional photo by modifying the Hero section HTML

Styling Changes
Colors: Modify CSS custom properties in the :root selector

Layout: Adjust grid and flexbox settings in respective CSS classes

Typography: Change font sizes and weights in the CSS
